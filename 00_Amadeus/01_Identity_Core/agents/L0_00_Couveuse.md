# L0_00 — Couveuse des Vaisseaux‑Monde

## Rôle
La couveuse est un espace de **gestation souveraine** où naissent et mûrissent les vaisseaux‑monde.
Elle protège le **Céleste A0** des tâches techniques afin qu’il puisse occuper son rôle d’**Observateur Type‑4 (Kardashev, solarpunk)**.

## Hiérarchie fonctionnelle
- **A1 (Gatekeeper)** : Visionnaire E‑Myth, filtre les missions et protège le cap.
- **A2 (Managers)** : structurent, orchestrent, stabilisent.
- **A3 (Techniciens)** : exécutent, testent, documentent.

## Couches
- **L0** : fondation / infra souveraine
- **L1** : vie / équilibre / exécution
- **L3** : expansion / projection / colonisation

## Principe d’or
A0 observe. La Couveuse exécute.
