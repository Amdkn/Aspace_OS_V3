# Instructions pour créer un Google Spreadsheet à partir des données CSV

## 1. Créer un nouveau Google Spreadsheet

1. Allez sur [Google Drive](https://drive.google.com)
2. Cliquez sur le bouton "+ Nouveau" dans le coin supérieur gauche
3. Sélectionnez "Google Sheets" dans le menu déroulant

## 2. Importer les données CSV

### Méthode 1 : Importation directe

1. Dans votre nouveau Google Spreadsheet, cliquez sur "Fichier" dans la barre de menu
2. Sélectionnez "Importer"
3. Choisissez l'onglet "Téléverser"
4. Faites glisser le fichier CSV "SBO LIST WEBSITE- CONTACT.csv" ou cliquez pour le sélectionner
5. Dans les options d'importation :
   - Sélectionnez "Remplacer la feuille de calcul"
   - Vérifiez que "Détecter automatiquement" est sélectionné pour le séparateur
   - Cliquez sur "Importer"

### Méthode 2 : Copier-coller

Si vous avez déjà les données dans un format texte :

1. Ouvrez votre nouveau Google Spreadsheet
2. Sélectionnez la cellule A1
3. Collez les données que vous avez copiées du fichier CSV

## 3. Formatage et organisation

Une fois les données importées, vous pouvez :

1. Figer la première ligne (en-têtes) :
   - Cliquez sur "Affichage" > "Figer" > "1 ligne"

2. Mettre en forme le tableau :
   - Sélectionnez toutes les données
   - Cliquez sur "Format" > "Mise en forme comme un tableau"
   - Choisissez un style qui vous convient

3. Ajuster la largeur des colonnes :
   - Double-cliquez sur les séparateurs entre les en-têtes de colonnes pour ajuster automatiquement

4. Filtrer les données :
   - Sélectionnez la ligne d'en-tête
   - Cliquez sur "Données" > "Créer un filtre"

## 4. Sauvegarde et partage

1. Renommez votre feuille de calcul en cliquant sur le titre en haut (par défaut "Sans titre")
2. Pour partager, cliquez sur le bouton "Partager" dans le coin supérieur droit
3. Entrez les adresses e-mail des personnes avec qui vous souhaitez partager le document
4. Définissez les permissions (modification, commentaire ou lecture seule)
5. Cliquez sur "Terminé"

Votre Google Spreadsheet contient maintenant toutes les données du fichier CSV "SBO LIST WEBSITE- CONTACT.csv" et est prêt à être utilisé.
