## 🚀 **Installation en 3 Étapes**

### **ÉTAPE 1 : Installer l'Extension Chrome** (2 min)

**Option A : Via Chrome Web Store** ⭐ RECOMMANDÉ

1. Va sur le Chrome Web Store :
   ```
   https://chromewebstore.google.com/detail/browser-mcp-automate-your/bjfgambnhccakkhmkepdoekmckoijdlc
   ```

2. Clique sur **"Ajouter à Chrome"**

3. Clique sur **"Ajouter l'extension"**

4. **Épingle l'extension** pour un accès facile :
   - Clique sur l'icône puzzle 🧩 dans la barre d'outils
   - Trouve "Browser MCP"
   - Clique sur l'icône épingle 📌

**Option B : Installation Manuelle (si pas disponible)**

```bash
# Cloner le repo
git clone https://github.com/BrowserMCP/mcp.git
cd mcp

# Charger l'extension dans Chrome
# 1. Va sur chrome://extensions/
# 2. Active "Mode développeur"
# 3. Clique "Charger l'extension non empaquetée"
# 4. Sélectionne le dossier de l'extension
```

---

### **ÉTAPE 2 : Configurer le Serveur MCP dans Claude Desktop** (3 min)

**2.1 - Ouvrir la Configuration**

1. Lance **Claude Desktop**
2. Va dans **Settings** (⚙️)
3. Clique sur l'onglet **Developer**
4. Clique sur **"Edit Config"**

Cela ouvrira le fichier `claude_desktop_config.json` situé ici :
- **Windows** : `%APPDATA%\Claude\claude_desktop_config.json`
- **macOS** : `~/Library/Application Support/Claude/claude_desktop_config.json`

**2.2 - Ajouter la Configuration Browser MCP**

Ajoute cette configuration dans le fichier JSON :

```json
{
  "mcpServers": {
    "browser-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "@playwright/mcp@latest"
      ],
      "env": {}
    }
  }
}
```

**Si tu as déjà d'autres MCP servers** (Notion, Airtable, etc.), ajoute juste la section `"browser-mcp"` dans `"mcpServers"` :

```json
{
  "mcpServers": {
    "notion": {
      "command": "npx",
      "args": ["-y", "@notionhq/mcp-server-notion"]
    },
    "browser-mcp": {
      "command": "npx",
      "args": ["-y", "@playwright/mcp@latest"]
    }
  }
}
```

**2.3 - Sauvegarder et Redémarrer**

1. **Sauvegarde** le fichier (Ctrl+S / Cmd+S)
2. **Ferme complètement** Claude Desktop
3. **Relance** Claude Desktop

---

### **ÉTAPE 3 : Connecter l'Extension au Serveur** (1 min)

1. **Ouvre Chrome** sur une page (n'importe laquelle)

2. **Clique sur l'icône Browser MCP** dans la barre d'outils

3. Dans le popup de l'extension, clique sur **"Connect"**

4. Tu devrais voir : **"✅ Connected to MCP Server"**

## 🛠️ **Dépannage**

### **❌ Problème : "MCP Server Not Connected"**

**Solution** :
```bash
# Vérifie que Node.js est installé
node --version

# Si pas installé, installe Node.js depuis nodejs.org
```

### **❌ Problème : "Extension Not Connecting"**

**Solution** :
1. Ferme **tous les onglets Chrome**
2. Ferme **Claude Desktop**
3. Relance **Claude Desktop** puis **Chrome**
4. Reclique sur **"Connect"** dans l'extension

### **❌ Problème : "Command 'npx' not found"**

**Solution** :
```bash
# Sur Windows via PowerShell (Admin)
npm install -g npx

# Redémarre Claude Desktop
```

### **❌ Problème : Configuration JSON Invalide**

**Solution** :
- Vérifie que tu n'as pas de virgules en trop
- Vérifie que toutes les accolades sont fermées
- Utilise un validateur JSON : https://jsonlint.com/
