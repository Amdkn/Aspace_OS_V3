## §4. Meta-Harness pour A0 — Optimisation de la Conception

### 4.1 Qu'est-ce que le Meta-Harness (Stanford + SuperagenticAI)

Le Meta-Harness (IRIS Lab Stanford + SuperagenticAI) est un framework
d'**optimisation automatique du harness autour d'un modèle fixe**.

```
Modèle fixe (non touché) :  Claude Sonnet / Gemini Flash / etc.
                                    │
               ┌────────────────────┴─────────────────────┐
               │            HARNESS (ce qu'on optimise)    │
               │  • Quoi récupérer en contexte             │
               │  • Quoi mémoriser entre sessions          │
               │  • Comment construire le prompt           │
               │  • Quels outils activer                   │
               │  • Quels fichiers injecter                │
               └────────────────────────────────────────────┘
                                    │
                          Output de l'agent

Boucle de recherche :
  1. Proposer une variation du harness (ex: injecter LORE.md vs pas)
  2. Évaluer sur search set (N sessions de rédaction SDD)
  3. Mesurer le score (line_density_score + gate pass rate)
  4. Garder les meilleures variations
  5. Encoder en Skill Hermes Nous (mémoire durable)
```
