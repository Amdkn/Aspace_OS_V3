# Entity: ABC OS (Agricultural Business Concept OS)

> SaaS platform for agricultural formation — AI-first, VAPI-powered, Zod-validated.

ABC OS = **Agricultural Business Concept Operating System**. Un projet SaaS pour la formation agricole, construit sur les principes A'Space OS avec une interface vocale AI-first.

---

## Concept

Formation agricole = former les agriculteurs à meilleure pratiques, gestion de культуры, lecture de données. ABC OS fait ça via:
1. **VAPI** — interface vocale (parle à l'agriculteur en français)
2. **AI-first** — pas d'app mobile, juste la voix + un backend
3. **Zod contracts** — validation de toutes les données agricoles

---

## OMKSBS Doctrine

> Our Main Key Selling Business Story

ABC OS incarne le principle **OMKSBS**:
> AI-first, pas voice-first. La voix est un canal, pas le produit.

Le produit = formation agricole betterée. Le canal = VAPI.

## ABC OS dans A'Space OS

ABC OS est un **projet** sous le domaine SOA07 (Legal/Eternals) ou SOA05 (Growth/Guardians) selon le focus:

- **Revenue model**: formation payante → génère du cashflow
- **Legal**: contrats de formation, protection des données agricoles
- **Growth**: acquisition d'agriculteurs, pipeline
