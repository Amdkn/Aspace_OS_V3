# Entity: VAPI (Voice AI Infrastructure)

> VAPI = Voice AI Platform | vapi.ai | Used in ABC OS (Agricultural SaaS)

VAPI est l'**interface vocale AI-first** pour les projets SaaS dans A'Space OS, principalement utilisé dans [[entity_abc_os]] (Agricultural SaaS Formation).

---

## Usage dans A'Space OS

### ABC OS — Voice Interface

ABC OS (Agricultural Business Concept) utilise VAPI comme couche d'interface vocale :

```
Farmer → Voice (VAPI) → ABC OS → Crop Management
                    ↓
            AI Agent (Gemini/Claude)
                    ↓
            Zod Contracts (Sales/Finance/Legal)
```

### OMKSBS — AI-First Doctrine

VAPI incarne le principe **OMKSBS** (Our Main Key Selling Business Story):
> AI-first, pas voice-first. La voix est un canal, pas le produit.
