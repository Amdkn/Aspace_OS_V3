# Concept: L2 B1/B2/B3 — The Fractal Command Stack

> **Thèse** : dans L2 Business, le rang global A0–A3 est incarné en un **stack vertical B1/B2/B3**
> (Direction → Domaines → Exécution), et ce stack **se répète à deux échelles** : la Area perpétuelle
> de Jerry (macro) et les projets Picard de Summer (micro). C'est le « Fractal Engine ».

- **Macro** = `02_Areas_Spock/J01_Jerry_Prime_LD01_Business/` (B1_Area_Direction / B2_Area_Domains /
