## Frameworks

### Ikigai (Orville)

Framework japonais pour trouver son "raison d'être":
```
Ikigai = Ce que j'aime × Ce que je sais faire × Ce dont le monde a besoin × Ce pour quoi je peux être payé
```
- Japanese origin
- Used for: finding meaning in daily work
- In A'Space OS: gives the "why" behind A0's mission

### 12WY — 12-Week Year (SNW)

Brad Sugars method adapted for A'Space OS:
- 12 weeks = one execution cycle (not 52 weeks)
- Forces: Focus, urgency, measurement
- In A'Space OS: the execution cadence for all A2/A3 tasks

### PARA (Enterprise)

 Tiago Forte's Project/Areas/Resources/Archives:
- **P**rojects: active initiatives with deadlines
- **A**reas: ongoing responsibilities
- **R**esources: reference material
- **A**rchives: inactive stuff

Most mentioned framework in 2026-03 (326 mentions).

### GTD — Getting Things Done (Cerritos)

David Allen's system:
- Capture everything
- Clarify: is it actionable?
- Organize: project / someday / trash
- Reflect: weekly review
- Engage: do it

In A'Space OS: handles the chaos of the daily inbox.

### DEAL — Liberation (Protostar)

Custom framework for breaking old patterns:
- **D**efine: what pattern to break
- **E**xamine: why it exists
- **A**ct: replace with new pattern
- **L**ock: verify the new pattern holds

Used for: personal transformation, not execution.

### Life Wheel (Discovery)

Balance across 8 life domains:
- Career / Finance / Health / Family
- Social / Personal Growth / Fun / Environment

In A'Space OS: mirrors the 8-domain Solaris OS structure.
