## 2. ICP Solaris canon — 5 piliers

### Pilier 1 — Persona "Technicien fondateur E-Myth"

**Définition canon** (Takeout Gemini 2026-03 L3330-3340) : technicien (dev, designer, marketeur, intégrateur IA) qui a fondé une agence et se retrouve coincé entre production et direction. Il rêve de scaler mais vit dans le delivery au quotidien.

**4 sub-types** (origin spec verbatim) :
1. **Marketing Digital** — agence social media / SEO / ads sans stack technique profond
2. **Création** — agence créative / branding / identité visuelle sans delivery structuré
3. **Web** — agence web / dev / e-commerce avec dépendance sous-traitance
4. **AI Agencies** — agence qui revend expertise IA sans gouvernance ni conformité

**Démographie** : 28-45 ans, France + Europe francophone, CA agence 50K€ — 500K€, 2-5 personnes, TJM 400-800€ ou projet 5-50K€, stack Notion + Excel + Stripe (non intégré).

### Pilier 2 — Mantra doctrinal "L'illusion du sur-mesure"

**Verbatim ×3 occurrences** (Takeout Gemini 2026-05) :

| Line ref | Verbatim |
|---|---|
| `[2026-05:19068]` | "L'illusion du sur-mesure, c'est que chaque client pense avoir un cas unique alors que 80% des besoins sont standardisables" |
| `[2026-05:19213]` | "Le piège du technicien fondateur : croire que chaque projet mérite du sur-mesure. Faux. Le sur-mesure est l'ennemi de la scalabilité." |
| `[2026-05:25337]` | "Tarifer du sur-mesure quand on est solo founder, c'est se condamner à l'esclavage client. La solution AaaS : produit fini, config simple, prix fixe." |

**Implication AaaS** : Solaris vend un **produit fini AaaS** (Agent-as-a-Service), pas du consulting/dev sur-mesure. Le sur-mesure est **anti-pattern**.

### Pilier 3 — Marché haute entropie 136,1 Mds$

**TAM verbatim** (Takeout Gemini 2026-05 L155599) : **136,1 Mds$ marché intégrateurs système** 2026.

**4 sections étude de marché** (sister `ADR-MARKET-STUDY-001`) :
1. **La Mutation** — transition digitale accélérée post-2024
2. **L'Offre E-Myth** — agencies/web E-Myth = 80% des acteurs, pas de système
3. **Services Hybrides** — mix SaaS + services + AI agents 2026
4. **Les 3 Modèles** — pricing Solo Founder / Standard / Groupe (sister `ADR-AAAS-PRICING-001`)

**Caractérisation** : marché fragmenté, no dominant player, faible standardisation, marges compressées sur le sur-mesure. **Opportunité AaaS** = standardiser ce que le marché ne standardise pas.

### Pilier 5 — Killer Feature "Agentic Governance"

**4 mécanismes canon** (verbatim Takeout Gemini 2026-05 L156796-156844) :

1. **Action Space Bounding** — périmètre d'action strict par agent IA (no scope creep, no unauthorized actions)
2. **Sandboxing** — isolation des exécutions agents (no blast radius, no contamination)
3. **HITL Dynamique** — Human-In-The-Loop adaptatif selon criticité (auto-approve low-risk, manual-approve high-risk)
4. **Traçabilité AI-Act** — log complet décisions agent pour audit conformité (registre AI-Act article 12)

**Différenciateur marché** : aucun concurrent direct n'offre ces 4 mécanismes packagés en produit AaaS prêt-à-l'emploi (étude de marché 'The Builders' 2026).

**Driver légal** : AI-Act application totale **août 2026** (sister `ADR-MARKET-STUDY-001` §Driver légal).
