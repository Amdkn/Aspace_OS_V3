### 1.1 La Structure — Hanif (Réduction de Friction)

```
Source : "Business is hard until you build systems like this" (Shan Hanif)

Enseignement fondateur :
  On ne peut pas automatiser un processus cassé.
  La friction entre l'intention et l'exécution est le vrai problème.
  Un SOB ne naît pas d'un pivot — il naît d'une systématisation radicale.

Application SOB Factory :
  SOLARIS, NEXUS et ORBITER existent parce que leurs ICP cibles
  n'ont pas la capacité interne de systématiser leur propre chaos.
  Nous vendons la systématisation — pas l'outil.

Axiome Hanif :
  "Toute offre SOB doit réduire au moins 3 frictions critiques
   dans le quotidien de l'ICP avant même d'automatiser quoi que ce soit."
```

### 1.2 Le Carburant — Hormozi (IA comme Levier Concurrentiel)

```
Source : "How to Win With AI in 2026" (Alex Hormozi)

Enseignement fondateur :
  L'IA n'est plus une aide à la rédaction.
  C'est l'arme pour écraser la concurrence, réduire le CAC,

  des équipes entières pour des tâches remplaçables.

Application SOB Factory :
  Chaque ICP intègre nativement des agents IA dédiés

  L'acheteur ne paie pas pour du logiciel — il paie pour un AVANTAGE.

Axiome Hormozi :
  "Si l'IA ne réduit pas le CAC ou n'augmente pas les marges
   d'au moins 30%, elle n'est pas dans la bonne couche du système."
```

### 1.3 Le Véhicule — YC (L'OS comme Franchise B2B)


> **AaaS = Agency / Architecture as a Service.**
> On ne vend pas une prestation de service (l'ancien modèle agence).
> On vend un système autonome qui produit le service pour l'ICP.
> Le fondateur installe l'OS. L'OS fait tourner l'agence.

### 2.2 La Structure d'un OS Variant

```
Un OS Variant = un Summer's Verse Template pré-configuré.

À la livraison d'un OS Variant, le client reçoit :

  1. Le Context Pack ICP (SDD-007 §5) — brief de démarrage
  2. Les 7 Domaines DC pré-configurés pour son secteur
  3. Les 7 Squads Marvel avec leurs agents IA calibrés
  4. Un set de SOPs de base (Batman Ops Starter Pack)
  5. Les KPIs primaires configurés dans sa Scorecard 12WY
  6. Les automatisations n8n de base opérationnelles (Zero)
  7. Un accès au Sunday Uplink Ritual (template rapport)

Ce qui n'est PAS livré :
  ✗ La vision stratégique (c'est Jerry du client)
  ✗ Le contenu métier (c'est l'expertise du client)
  ✗ La décision d'exécution (c'est Summer du client)
  → Le SOB Factory vend l'INFRASTRUCTURE DÉCISIONNELLE, pas les décisions.
```
