# Mode Orbiter — field-first / logistics / SOP-replicable / (sometimes compliance-bound)

**Books referenced (Area doctrine):** Built to Sell (Warrillow), E-Myth Revisited, Who Not How.
**Buyer:** field/operational; trusts **proof of reliability**, not promises. The North Star is usually a
**recurring contract** or **qualified inquiry**, not a one-off job. Quality must come from the **SOP/system**,
not from one person.

## Calibration knobs

| Knob | Orbiter value |
|---|---|
| North Star (Superman sets) | e.g. "Orbiter-fit recurring contracts from reliability/proof-led non-paid channels" or "Orbiter-fit qualified inquiries (compliance-safe)" |
| JTBD-002 rejection theme | **route-density / recurring-contract profitability** (field-service) or **structure/compliance-readiness** (regulated) — reject one-off jobs that don't densify the route, or operators not ready to run the SOP |
| JTBD-003 message angle | **reliability operated by SOP** + **visible/proven result** (timestamped checklist, before/after photos, presence tracking). Sell the system, not the hero |
| JTBD-004 channel | local SEO intention (P14) + proof-of-quality content (before/after, public standard checklist) + client referral loop (P12) + recurring-quote opt-in (P15) |
| JTBD-004 lag indicator | "≥1 Orbiter-fit recurring contract signed" or "≥1 compliance-safe qualified inquiry" — recurring/retention, not one-off |

## Rejection-criteria seeds
- R1 — one-off job that doesn't densify the route → unprofitable logistics.
- R2 — operator/structure not SOP-ready (regulated: not compliance-ready) → can't deliver the promise.
- R3 — wants a price race to the bottom → erodes margin, P10 churn.

## Painkiller variant seeds
- V1 — constant standard ("the same standard every visit, whoever holds the broom").
- V2 — guaranteed presence ("tracked presence, zero no-show").
- V3 — proven quality ("every visit proven — checklist + photo; quality you see, not promised").

## Compliance-bound Orbiter
If the project is regulated (childcare, health, etc.), it is **Legal-gated** — see `special-cases.md`.
Never let a variant promise something that contradicts regulation, and route every public message
through Legal before diffusion. Drax additionally kills any promise the Ops/regulation can't honor.
