# Mode Solaris — visual-first / brand-led

**Books referenced (Area doctrine):** $100M Offers (Hormozi), StoryBrand (Miller).
**Buyer:** visual-first, scrolls brand content, "buys on portfolio/samples".

## Calibration knobs

| Knob | Solaris value |
|---|---|
| North Star (Superman sets) | "Solaris-fit demos booked from brand/visual-led non-paid channels" |
| JTBD-002 rejection theme | **visual-first markers** — reject prospects who can't/won't judge on aesthetics, want lowest-bid commodity work, or have no brand to elevate |

| JTBD-004 lag indicator | "≥1 Solaris-fit demo booked from the piece" |

## Rejection-criteria seeds (adapt to the actual JTBD-001 pains)
- R1 — price-led commodity buyer (judges on cost, not transformation) → wrong economics.
- R2 — no existing brand/visual surface to elevate → offer has no leverage.
- R3 — wants endless revisions without a decision-maker → unbounded effort, P10 churn risk.

## Painkiller variant seeds
- V1 — transformation visible ("your brand finally looks like the leader you are").
- V2 — proof over promise (showcase / before-after portfolio).
- V3 — guide framing (you bring the vision, we engineer the craft).
