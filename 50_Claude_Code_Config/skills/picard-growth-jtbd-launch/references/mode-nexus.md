# Mode Nexus — data-first / expertise / knowledge-community

**Books referenced (Area doctrine):** Expert Secrets (Brunson), The Consulting Bible, E-Myth Revisited.
For **community** variants also: Group Genius (network effect, belonging, retention).
**Buyer:** evaluates on expertise/proof; for community Nexus, the North Star is an *active returning member*, not a "lead".

## Calibration knobs

| Knob | Nexus value |
|---|---|
| North Star (Superman sets) | Expertise variant: "Nexus-fit discovery calls booked from expertise/proof-led non-paid channels". Community variant: "Nexus-fit member activations (joined + active in week 1)" |
| JTBD-002 rejection theme | **operator-readiness** (expertise) or **signal-density** (community) — reject mass-admission / tire-kickers; keep the dense-signal few (P11) |
| JTBD-003 message angle | **Expert Secrets** framing — authority + new opportunity; for community add **belonging + accountability** |
| JTBD-004 channel | proof-content (teardown/case data) + opt-in (P15); community: activation ritual in week 1. Apollo/LinkedIn scrape, if proposed, is **GATED on A0 sign-off + Apollo key** (never auto-run) |
| JTBD-004 lag indicator | Expertise: "≥1 Nexus-fit discovery call booked". Community: "≥1 member activation (joined + active week 1) — retention, not signups" |

## Rejection-criteria seeds
- R1 — wants a vitamin, not a painkiller (curious but no operating pain) → no urgency, P8 fail.
- R2 — mass-admission mindset / low signal density (community) → dilutes the network, P10 churn.
- R3 — won't engage with proof/expertise (price-only) → wrong fit for an expertise offer.

## Painkiller variant seeds
- V1 — authority/new-opportunity ("the operators who solved X did it this way").
- V2 — accountability/belonging (community) ("you stop learning alone").
- V3 — proof-led ("here's the data, not a pitch").

## Community-specific note
For knowledge-community Nexus (e.g. members space), the graduation gate is **active-member retention**,
not signups. Make the lag indicator a returning/active metric and say so explicitly.
