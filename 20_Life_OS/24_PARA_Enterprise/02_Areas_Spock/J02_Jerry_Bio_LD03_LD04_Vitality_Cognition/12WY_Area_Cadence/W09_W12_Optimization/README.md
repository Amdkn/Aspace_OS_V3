# W09–W12: Summer-Autumn/Resilience Testing

## Quarter Focus

**Purpose**: Maintain stability during summer travel season, stress-test the biological stack, pre-winter protocol preparation.

**Key Insight**: Summer is the easiest season biologically (HRV peaks, sleep is natural, light is abundant) — but it is also the most disruptive season structurally (travel, schedule changes, vacations). This quarter is about maintaining the spring gains through the chaos of summer while preparing for winter.

---

## W09: Pre-Summer Planning

### Focus

```
→ Identify summer ORANGE risk factors
→ Plan for travel disruption protocols
→ Set up pre-winter strength base
```

### Key Actions

```
□ Summer risk assessment:
  → Travel schedule: When, where, how long?
  → Schedule disruptions: Vacations, work trips, family visits?
  → ORANGE risk during disruption: What will be hardest to maintain?
    → Sleep window consistency?
    → Zone 2 150min?
    → HRV monitoring?

□ Travel protocol preparation:
  → Pack list: Wearable, resistance bands, cold exposure gear
  → Sleep optimization: Travel pillow, eye mask, earplugs
  → Zone 2 alternatives: Bodyweight circuits, swimming, walking
  → Learning: Download Anki decks, pre-plan learning blocks
  → If international: Time zone adjustment protocol

□ Strength base check:
  → Is current strength level sustainable?
  → Any ROM deficits that need addressing before winter?
  → Pre-winter base building begins W10

□ Pre-winter vitamin D plan:
  → Summer sun will restore vitamin D — confirm levels W09
  → If vitamin D ≥50ng/mL: Note, maintain through fall
  → If vitamin D <50ng/mL: Plan supplementation for Oct start
```

### Summer Biology vs Winter Biology

```
| Domain | Summer Peak | Winter Low | Implication |
|--------|-------------|------------|-------------|
| HRV | Highest (+15% vs winter) | Lowest | Summer is resilience peak |
| Sleep | Easiest (longer nights) | Hardest | Capitalize on sleep ease |
| Vitamin D | Peak (sun exposure) | Lowest | Build summer stores |
| Zone 2 | Outdoor easy | Indoor required | Maximize outdoor training |
| Cold Exposure | Hardest (heat) | Easiest | Modify to cold shower |
```
