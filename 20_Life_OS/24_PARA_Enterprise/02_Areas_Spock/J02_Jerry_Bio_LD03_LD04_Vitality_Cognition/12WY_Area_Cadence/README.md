# 12WY — Jerry Bio: Area Cadence
## 12-Week Seasonal Biological Rhythm

> **Critical Distinction**: J02 cadence is NOT about growth.
> It is about CONSTANCY. The 12WY cycles here track:
> - Seasonal biological variations (winter vs summer HRV, sleep patterns)
> - Annual biological rhythms (year-over-year trend analysis)
> - Long-term substrate trajectory (are we trending toward RED or maintaining GREEN?)
>
> **Tone**: Constancy over optimization. The goal is to not get worse.
> Any improvement beyond maintaining GREEN is a bonus, not a target.

---

## 1. 12WY Cadence Philosophy

### 1.1 Why Seasonal for Biology?

```
Biological systems are not linear.
They have:
  → Circadian rhythms (24h)
  → Weekly rhythms (stress patterns)
  → Seasonal rhythms (HRV, sleep, vitamin D, cold exposure)
  → Annual rhythms (fitness cycles, learning velocity cycles)

Business cadence (LD01) = optimize for growth
Biological cadence (J02) = optimize for stability
  → Same baseline maintained across seasons
  → Seasonal adjustments prevent ORANGE drift
  → Annual rhythm analysis catches long-term degradation
```

### 1.2 The Three Phases

```
W01–W04: WINTER/FOUNDATIONAL (Dec–Mar or analogous seasonal anchor)
  → Purpose: Establish baseline, review prior year, account for winter biology
  → Key concern: Lower light = lower HRV, lower vitamin D, more sleep debt
  → Adjustment: Longer sleep targets, higher Zone 2 volume, vitamin D supplementation

W05–W08: SPRING/OPTIMIZATION (Mar–Jun or analogous)
  → Purpose: Fine-tune protocols from winter data, optimize for spring/summer
  → Key concern: Outdoor training, sleep window adjustment, Zone 5 reintroduction
  → Adjustment: Earlier wake times, outdoor Zone 2, increased cold exposure

W09–W12: SUMMER-AUTUMN/RESILIENCE TESTING (Jun–Sep or analogous)
  → Purpose: Stress-test the stack before year-end, prepare for winter
  → Key concern: Heat adaptation, consistency despite travel, pre-winter strength base
  → Adjustment: Maintain training, pre-winter protocol preparation
```

## 2. Annual Biological Calendar

### 2.1 Seasonal Biological Variations

```
WINTER BIOLOGY (Dec–Feb):
  → HRV typically 10–15% lower than summer baseline
  → Sleep requirement typically 30–60min longer
  → Vitamin D levels at seasonal low
  → Cold exposure more natural (outdoor cold easy)
  → Risk: HRV ORANGE if summer baseline was not established

SPRING BIOLOGY (Mar–May):
  → HRV recovering (light exposure increasing)
  → Energy rising (circadian alignment improving)
  → Training volume can increase
  → Risk: Overexertion after winter rest

SUMMER BIOLOGY (Jun–Aug):
  → HRV peaks (longest days, most light)
  → Sleep typically easier (warm, long evenings)
  → Risk: Travel disruption, heat stress, inconsistent schedule
  → Opportunity: Best time for learning velocity experiments

AUTUMN BIOLOGY (Sep–Nov):
  → HRV declining (light decreasing)
  → Pre-winter strength base building
  → Risk: Back-to-school stress, pre-holiday overcommitment
  → Preparation: Winter protocol setup, vitamin D restocking
```

### 2.2 Year-Over-Year Analysis

```
Annual Review Purpose:
  → Is the biological baseline improving or degrading year-over-year?
  → Which domains consistently underperform in which seasons?
  → Has the Founder Load Ceiling changed (improved or degraded)?
  → Beth HALT events: Year 1 vs Year 2 vs Year 3 trajectory

Target Trajectory:
  → Year 1: All GREEN, no RED, ORANGE events declining each quarter
  → Year 2: All GREEN, no RED, ORANGE events <1/month
  → Year 3: Predictive ORANGE (detected before subjective awareness)
```

## 3. 12WY Quarter Structure

### 3.1 W01–W04: Winter/Foundational

```
FOCUS: Establish winter baseline, adjust for seasonal biology

Key Activities:
  → Week 1: Review prior year's annual biological audit
  → Week 2: Establish winter baseline metrics (HRV, sleep, cognitive load)
  → Week 3: Implement winter protocol adjustments (sleep +30min, Zone 2 +20min)
  → Week 4: Mid-winter checkpoint — are metrics stable?

Winter Adjustments:
  → Sleep: Target 7.5–8h (vs 7h standard)
  → Zone 2: Increase 20min if HRV allows
  → Vitamin D: Test levels Oct–Mar, supplement if <40ng/mL
  → Cold exposure: Outdoor exposure prioritized (natural cold)
  → HRV: Expect 10–15% lower than summer — adjust GREEN threshold temporarily
```

### 3.2 W05–W08: Spring/Optimization

```
FOCUS: Fine-tune based on winter data, capitalize on seasonal energy

Key Activities:
  → Week 5: Analyze winter data — what worked, what didn't
  → Week 6: Optimize sleep window (earlier wake as light increases)
  → Week 7: Reintroduce Zone 5 sprints (if discontinued in winter)
  → Week 8: Spring biological audit — full domain review

Spring Adjustments:
  → Sleep: Can reduce to 7h if HRV is GREEN and stable
  → Zone 2: Outdoor sessions as weather allows
  → Zone 5: Resume 1×/week if HRV has recovered
  → Learning: Best quarter for new domain acquisition (cognitive energy high)
```

### 3.3 W09–W12: Summer-Autumn/Resilience Testing

```
FOCUS: Maintain stability during travel season, pre-winter preparation

Key Activities:
  → Week 9: Pre-fall planning — identify travel/holiday ORANGE risks
  → Week 10: Stress-test protocols with reduced schedule flexibility
  → Week 11: Pre-winter protocol preparation (vitamin D, sleep window)
  → Week 12: Annual biological audit, set baseline for next year

Autumn Adjustments:
  → Sleep: Begin extending window (+15–30min before winter)
  → Zone 2: Indoor sessions prioritized as weather shifts
  → Cold exposure: Increase to 3×/week for BAT maintenance
  → Strength: Pre-winter base building (injury prevention for winter sports)
```

## 5. Cadence Review Protocol

### 5.1 Week-End Review (5min)

```
Week-End Check (every Sunday):
  → All 8 domains GREEN? (Y/N — if N, ORANGE event logged)
  → Any lead metric adherence <80%? (Y/N — if Y, root cause)
  → Any recovery protocols in progress? (Y/N — if Y, track to completion)
  → Any Beth notifications required? (Y/N — if Y, send within 24h)
```

### 5.2 Quarter-End Review (60min)

```
Quarter-End Review (end of W04, W08, W12):
  → All ORANGE/RED events reviewed (root cause analysis)
  → Lead metric adherence averaged (target >80%)
  → Lag metric trajectory analyzed (improving/stable/declining)
  → Next quarter adjustments planned
  → Any protocol changes documented and communicated
```
