# B2 Domain 02 — Breath/Oxytocin

## Domain Identity

### What It Protects

Breath/Oxytocin is the **autonomic regulation domain** — it directly controls:
- Parasympathetic (rest-and-digest) nervous system activation
- CO2 tolerance capacity (resilience to stress)
- Heart rate variability (HRV) via vagal tone
- Sleep onset latency (breath pattern entering sleep)
- Emotional regulation during high-cognitive-load periods
- Recovery from sympathetic (fight-or-flight) activation

### Why It Matters

```
Nasal breathing → 50% more CO2 retention → better O2 delivery to tissues
Mouth breathing → reduced CO2 tolerance → higher baseline anxiety
Box breathing (4-6 breaths/min) → HRV increase 15–25% within 5min
CO2 tolerance training → reduced panic response to stress
Exhale:inhale ratio 2:1 → vagal activation → faster recovery from ORANGE
```

## Jerry's Stewardship Protocol

### Daily Practice (Non-Negotiable)

```
Morning (upon waking):
  □ 5min nasal breathing meditation — box breathing 4-4-4-4 or 4-6-4

Throughout day:
  □ CO2 tolerance training: 30s breath hold after exhale, 4–5×/day
  □ Exhale:inhale ratio check: exhale twice as long as inhale during stress
  □ Nasal breathing default — mouth tap reminder if mouth breathing detected

Evening (pre-sleep):
  □ 5min nasal breathing cooldown
  □ Box breathing if HRV is declining
```

### Monitoring

```
Daily:
  □ Breath hold duration (morning, after exhale)
  □ Nasal vs oral breathing ratio (subjective check each hour)
  □ Exhale:inhale ratio during rest (target: 2:1 minimum)

Weekly:
  □ CO2 tolerance trend (morning breath hold over 7 days)
  □ HRV trend correlation (breath → HRV should be visible in 48h)
```

## Escalation Rules

```
ORANGE Trigger: Breath hold <25s for 3+ consecutive days OR exhale:inhale <1.8:1 for 3+ days
  → Check sleep quality (breath issues often indicate sleep apnea or nasal obstruction)
  → Implement nasal dilation (breath strips, nasal dilator) if structural
  → Increase box breathing practice to 10min/night
  → Log in Jerry Bio scorecard with root cause

RED Trigger: Breath hold <15s OR chronic mouth breathing >50% of sleep
  → Medical referral recommended (sleep study for apnea)
  → HARD FREEZE if correlated with HRV RED
  → Beth notification if HRV RED + Breath RED simultaneously
```

---

## ORANGE Recovery Protocol

### Immediate Actions (within 24h)

```
1. Nasal breathing audit: identify obstruction (allergies, congestion, deviated septum)
2. Implement nasal dilation protocol: breath strips + saline spray + humidifier
3. Increase box breathing to 10min morning + 10min evening
4. Eliminate mouth tape if currently used (addresses symptom not cause)
5. Add CO2 tolerance holds: 4× daily, 30s each
6. Reduce evening alcohol (relaxes throat muscles → mouth breathing)
7. Elevate head of bed 4–6 inches if supine mouth breathing suspected
```
