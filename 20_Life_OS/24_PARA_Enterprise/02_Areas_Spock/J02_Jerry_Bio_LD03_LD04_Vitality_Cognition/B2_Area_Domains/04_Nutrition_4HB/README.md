# B2 Domain 04 — Nutrition/4HB

## Domain Identity

### What It Protects

Nutrition/4HB is the **metabolic optimization domain** — it protects:
- Blood glucose stability (cognitive performance, mood regulation)
- Inflammatory baseline (chronic low-grade inflammation degrades cognition)
- Gut microbiome diversity (gut-brain axis, immune function)
- Nutrient timing around exercise (recovery, adaptation)
- Body composition (metabolic rate maintenance)
- Ketone production capacity (cognitive fuel flexibility)

### Why It Matters

```
Glucose stability → cognitive performance maintained 4–6h post-meal
Inflammatory foods → degraded learning velocity (IL-6 impairs memory consolidation)
Gut microbiome → 90% of serotonin produced in gut (mood regulation)
Nutrient timing → 30% better recovery with protein within 30min post-exercise
Omega-3 (fish oil) → BDNF support (learning velocity)
Vitamin D → HRV correlation (winter: check levels more frequently)
```

## Jerry's Stewardship Protocol

### Metabolic Baseline

```
Daily:
  □ 2–3 main meals, no snacking between (intermittent feeding window)
  □ Protein within 30min post-exercise (30–40g)
  □ Fat + fiber with each meal (glucose stability)
  □ Minimum 2.5L water

Weekly:
  □ Omega-3: 2+ servings fatty fish (salmon, sardines, mackerel)
  □ Vegetables: 5+ servings/day average
  □ Processed food: <10% of total calories

Quarterly:
  □ Blood work: fasting glucose, HbA1c, lipid panel, inflammatory markers
  □ Vitamin D level (especially Oct–Mar)
```

### Food Quality Rules

```
Priority 1 — Anti-inflammatory staples:
  → Leafy greens, cruciferous vegetables (5+ servings/day)
  → Fatty fish 2×/week (omega-3)
  → Extra virgin olive oil (polyphenols)
  → Berries (antioxidants)
  → Nuts/seeds (omega-3, fiber)

Priority 2 — Metabolic stability:
  → Protein at every meal (0.8–1g/lb bodyweight)
  → Fiber: 30g+/day minimum
  → Starch: post-exercise only, minimal otherwise

Avoid (ORANGE triggers):
  → Added sugar >25g/day
  → Refined carbs >100g/day
  → Trans fats (any amount)
  → Industrial seed oils (soybean, corn, canola)
```

## Escalation Rules

```
ORANGE Trigger: Glucose crashes 3+ times/week OR processed food >25% for 1+ week
  → Implement food logging (MyFitnessPal or equivalent) for 2 weeks
  → Identify trigger foods (usually: refined carbs, alcohol, missing protein)
  → Remove one trigger food category for 1 week (not all — isolation required)
  → If CGM available: review weekly patterns

ORANGE Trigger: Alcohol >8 drinks/week
  → Hard limit: maximum 4 drinks/week during ORANGE
  → No more than 2 drinks in any single day
  → 3+ consecutive days without alcohol required to clear ORANGE

RED Trigger: Daily glucose >140mg/dL fasting OR <70mg/dL at any point
  → Medical referral recommended
  → HARD FREEZE if correlated with HRV ORANGE
  → Beth notification if compound ORANGE (Nutrition + any LD03)
```

## ORANGE Recovery Protocol

### Immediate Actions (within 24h)

```
1. Food log for 1 week (identify triggers)
2. Remove most inflammatory food (usually: alcohol, refined carbs, or seed oils)
3. Add one anti-inflammatory food (usually: leafy greens, fish, olive oil)
4. Implement 16:8 feeding window (helps glucose stability)
5. Increase protein at breakfast (stabilizes glucose all day)
6. Eliminate snacks between meals (tests glucose stability)
7. Schedule quarterly blood work if not done recently
```
