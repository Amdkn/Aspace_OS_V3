# B2 Domain 03 — Movement/Outlive

## Domain Identity

### What It Protects

Movement/Outlive is the **cardiovascular and metabolic resilience domain** — it protects:
- VO2max (aerobic capacity — strongest predictor of longevity)
- Zone 2 aerobic base (fat oxidation, metabolic flexibility)
- Zone 5 sprint capacity (cognitive blood flow, HGH production)
- HRV resilience (exercise-induced stress → recovery adaptation)
- Metabolic rate (muscle mass, insulin sensitivity)
- Cognitive blood flow (acute exercise → 30min improved cognition)

### Why It Matters

```
VO2max > 35 mL/kg/min → Jerry Bio clearance for high-cognitive-load activities
Zone 2 training → HRV improvement within 4–6 weeks
Zone 5 sprints → BDNF release → learning velocity supported
Movement → sleep quality improvement (15–20% better sleep with morning exercise)
Sedentary > 8h → HRV decline even with 1h exercise (inactivity damage cumulative)
```

## Jerry's Stewardship Protocol

### Weekly Exercise Structure

```
Zone 2 Base (Priority 1):
  □ Minimum 150min/week at 60–70% max HR (fat oxidation zone)
  □ Can be split: 30min × 5 days or 50min × 3 days
  □ Best time: morning (sets circadian, improves HRV, aids sleep)

Zone 5 Sprints (Priority 2):
  □ Minimum 1×/week — 4–6 × 30s all-out sprints with 4min recovery
  □ Best time: afternoon or post-strength (not before sleep)
  □ Can be: cycling, rowing, running, or stair climbing

Resistance Training (Priority 3):
  □ Minimum 2×/week full-body or upper/lower split
  □ 10–20 working sets/week per muscle group
  □ Progressive overload: 1 variable (weight/reps/sets) progressed weekly
```

### Daily Monitoring

```
Every morning:
  □ AM HRV (Oura/Whoop/Apple Health) — primary signal
  □ RHR deviation from baseline

Every evening:
  □ Zone 2 minutes logged (if trained)
  □ Daily steps (minimum 7,000)
  □ Posture resets (minimum 4 — every 30min seated)

Weekly:
  □ VO2max estimated (if wearable) or quarterly formal test
  □ HRV trend over 7 days
  □ Zone 5 completion
```

## Escalation Rules

```
ORANGE Trigger: Zone 2 <150min OR steps <5,000 for 3+ days
  → ORANGE ADVISORY issued
  → Increase Zone 2 by 30min/week until GREEN
  → Reduce sitting: stand/walk every 25min (not optional)
  → Log root cause — usually schedule conflict or travel

ORANGE Trigger: HRV 60–64ms for 3+ days
  → Check sleep debt (most common cause)
  → Reduce Zone 2 intensity by 30%
  → Increase sleep window by 30min
  → Monitor for ORANGE cascade (Breath, Sleep)

RED Trigger: Exercise = 0×/week OR HRV <60ms
  → HARD FREEZE on all Jerry areas
  → Beth HALT notification within 24h
  → Recovery protocol: sleep first, then Zone 2 resumption at 50% load
  → No Zone 5 until HRV returns to GREEN
```

## ORANGE Recovery Protocol

### Immediate Actions (within 24h)

```
1. Identify primary stressor: HRV decline OR exercise gap
2. If HRV ORANGE: extend sleep window + reduce intensity (not eliminate movement)
3. If exercise gap: schedule minimum 2 Zone 2 sessions this week (non-negotiable)
4. Add walking meetings (not sitting meetings) for any non-essential calls
5. Install standing desk reminder: every 25min
6. Reduce evening screen time → reallocate to 20min evening walk
7. Log all metrics — ORANGE recovery requires behavioral evidence
```
