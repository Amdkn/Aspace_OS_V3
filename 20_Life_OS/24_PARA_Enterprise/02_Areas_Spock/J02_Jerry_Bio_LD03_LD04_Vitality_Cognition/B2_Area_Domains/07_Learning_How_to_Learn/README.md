# B2 Domain 07 — Learning/How to Learn

## Domain Identity

### What It Protects

Learning/How to Learn is the **cognitive acquisition infrastructure domain** — it protects:
- Active recall capacity (retrieval vs re-reading)
- Spaced repetition adherence (memory consolidation across time)
- Schema formation (chunking new info into existing knowledge)
- Interleaving practice (cross-domain skill development)
- Learning environment quality (low distraction = high signal)
- Working memory efficiency (3–4 chunk limit enforcement)

### Why It Matters

```
Active recall → 50% better retention vs re-reading (Oakley/Sejnowski)
Spaced repetition → 80% retention at 60-day mark vs 20% with massed practice
Interleaving → 43% better transfer to new problems vs blocked practice
Working memory limit → >4 chunks = cognitive overload = no learning occurring
Sleep-dependent consolidation → new schemas formed during sleep only
Desirable difficulty → optimal struggle zone produces 2–3× better retention
```

## Jerry's Stewardship Protocol

### Daily Learning Protocol

```
Learning Block Structure (45–90min optimal):
  1. Pre-learning (5min):
     → Rate cognitive load 1–10
     → Rate emotional state 1–10 (anxiety >7 = abort session)
     → Clear environment (phone airplane, browser closed)

  2. During learning (core):
     → Active recall: ≥60% of time (flashcards, free recall, practice problems)
     → Spaced repetition: apply to any concept encountered
     → Interleaving: switch domains every 25–30min (prevent focal blindness)

  3. Post-learning (5min):
     → Rate perceived difficulty 1–10
     → Log: concept acquired, time spent, method used
     → Schedule 24h retrieval practice

Spaced Repetition Minimum:
  □ New concept → first retrieval within 24h
  □ Second retrieval → 3-day interval
  □ Third retrieval → 7-day interval
  □ Fourth retrieval → 21-day interval
  □ All retrievals logged in Anki or equivalent
```

### Weekly Learning Review

```
Sunday 30min (non-negotiable):
  □ Review week's learning velocity (concepts acquired vs target)
  □ Check spaced repetition queue (all due cards completed)
  □ Assess interleaving adherence (did you practice >1 domain?)
  □ Calculate: learning progress vs 10-week project target
  □ Identify: what worked, what didn't (methodology, not content)
```

## Escalation Rules

```
ORANGE Trigger: Active recall <40% for 3+ learning sessions
  → ORANGE ADVISORY: re-reading is noise, not signal
  → Implement: flashcard creation for all new concepts
  → Eliminate: passive video lectures (use 1.5–2× speed + notes + recall)

ORANGE Trigger: Spaced repetition <60% adherence for 1+ week
  → ORANGE ADVISORY: spaced repetition is non-negotiable for retention
  → Implement: daily Anki review before new learning
  → All new concepts must be added to spaced repetition system

ORANGE Trigger: Learning blocks <20min average for 3+ days
  → Check: LD03 substrate (sleep debt is most common cause)
  → Check: cognitive load (is Jerry overcommitted?)
  → Reduce: new content until 45min blocks restored

RED Trigger: No new schema formed in 7+ days OR anxiety >8 during all learning attempts
  → HARD FREEZE on new domain acquisition
  → Beth notification if correlated with LD03 ORANGE
  → Full learning break: maintenance only (spaced repetition of existing)
  → Root cause: sleep debt? emotional overload? cognitive burnout?
```

## ORANGE Recovery Protocol

### Immediate Actions (within 24h)

```
1. Identify barrier: environment (distraction), cognitive load, or LD03 substrate
2. If environment: implement strict protocol (phone airplane, browser closed, 25min timer)
3. If cognitive load: defer new learning, do only spaced repetition of existing
4. If LD03 substrate: see Sleep/Recovery recovery protocol first
5. Add: pre-learning emotional check (abort if anxiety >7)
6. Implement: 2-day recovery learning (only easy review content, no new acquisition)
7. Resume new learning only after 48h of GREEN metrics
```
