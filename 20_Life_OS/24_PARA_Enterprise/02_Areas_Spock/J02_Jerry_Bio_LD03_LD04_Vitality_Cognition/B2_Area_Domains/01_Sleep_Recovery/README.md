# B2 Domain 01 — Sleep/Recovery

## Domain Identity

### What It Protects

Sleep/Recovery is the **keystone domain** — it is the primary substrate for:
- Hippocampal memory consolidation
- Cognitive waste clearance (glymphatic system)
- Emotional regulation (amygdala control)
- Learning velocity (LD04 downstream)
- HRV autonomic recovery
- Immune system function

**No domain operates independently of sleep. Sleep ORANGE cascades to HRV within 48h, to cognition within 72h.**

### Why It Matters

```
Sleep <6h for 3 nights → decision quality equivalent to legal intoxication
Sleep <5h for 1 night → cognitive bandwidth reduced by 25% next day
Sleep debt accumulation → HRV decline → autonomic resilience collapse
Sleep debt + business stress → highest-probability founder failure mode
```

## Jerry's Stewardship Protocol

### Monitoring

```
Daily (morning):
  □ Record: Actual sleep duration (hours:minutes)
  □ Record: Sleep onset latency (minutes)
  □ Record: Nightly awakenings (count + duration if >15min)
  □ Record: Subjective quality (1–10)
  □ Calculate: Sleep debt accumulation (7h baseline × nights)

Weekly (Sunday review):
  □ Calculate: Average nightly sleep duration
  □ Calculate: Total weekly sleep debt
  □ Assess: Trend over 4-week rolling window
  □ Identify: Any ORANGE precursors (late nights, debt accumulation)
```

### Escalation Rules

```
ORANGE Trigger: Any single GREEN threshold breached for >24h
  → Issue ORANGE ADVISORY within 24h

  → Log in Jerry Bio scorecard

RED Trigger: Sleep <5h for any night OR sleep <6h for 3+ consecutive nights
  → Issue HARD FREEZE to all Jerry areas immediately
  → Beth HALT notification within 24h
  → Full activity log to Beth within 48h
```

## ORANGE Recovery Protocol

### Mandatory Actions (within 24h)

```
1. Identify primary stressor — sleep debt, HRV decline, or external disruption
2. Remove one non-essential commitment (not defer — remove)
3. Extend sleep window by 30min minimum tonight
4. Mandatory 20min morning sunlight exposure (circadian HRV regulation)
5. Reduce evening cognitive load after 20:00
6. Cool room to 18–20°C if not already
7. Log all metrics in weekly scorecard
```

### Clearance Criteria

```
ORANGE clears only when:
  1. Target metric back to GREEN for minimum 48h consecutively
  2. All other Sleep/Recovery KRs also GREEN
  3. No new ORANGE signals introduced during recovery period
  4. Behavioral compliance demonstrated (not just metric compliance)
```
