# B2 Domain 06 — Strength/Supple Leopard

## Domain Identity

### What It Protects

Strength/Supple Leopard is the **structural integrity and movement quality domain** — it protects:
- Muscle mass (sarcopenia prevention, metabolic rate)
- Tendon/ligament resilience (injury prevention)
- Joint ROM and function (daily movement quality)
- Posture under load (seated work compensation)
- Injury recovery capacity (from Zone 2/Zone 5 training)
- Prolonged cognitive work capacity (physical structure supports sitting)

### Why It Matters

```
Resistance training → 30–50% better glucose metabolism (more than cardio alone)
Muscle mass → resting metabolic rate (1lb muscle = 6cal/day at rest)
Tendon resilience → injury prevention (Zone 5 sprint injury = RED for weeks)
Joint ROM → daily movement quality (hip flexor shortening = 30min less walking)
Posture → cognitive performance (forward head = reduced O2 to brain by 30%)
Morning strength → cortisol spike that aids focus for 2–3h post-training
```

## Jerry's Stewardship Protocol

### Resistance Training Structure

```
Minimum Standard (GREEN):
  □ 2×/week full-body OR upper/lower split
  □ 10–20 working sets per muscle group per week
  □ Progressive overload: weight OR reps OR sets increased weekly
  □ Primary compounds: squat, hinge, push, pull, carry

Optimal Standard:
  □ 3×/week if recovery allows (HRV GREEN)
  □ Single leg and unilateral work (anti-asymmetry)
  □ Load capacity work (strength > 8 reps = strength adaptation)
  □ Zone 5-compatible structure (no posterior chain fatigue before sprint day)

Weekly:
  □ Monday: Upper body push/pull
  □ Thursday: Lower body + carries
  □ (Saturday: optional third session if HRV allows)
```

### Daily Mobility Protocol

```
Kelly Starrett pre-rehab (mandatory if joint deficit):
  □ Hip flexor stretch 2×30s each side (if <90° hip ROM)
  □ Thoracic extension 2×30s (if <40° extension)
  □ Ankle dorsiflexion 2×30s each side (if <20°)
  □ Shoulder rotation 2×20 each direction (if ROM deficit)

General mobility (10–15min/day minimum):
  □ Morning: 5min movement-based activation (not static stretching)
  □ Midday: 5min standing desk protocol
  □ Evening: 5min post-work decompression

Posture resets (minimum 4/day):
  □ Every 30min seated → neck retraction, hip flexor stretch, standing
  □ Phone on airplane mode during meetings
  □ Walking meetings for any call <30min
```

## Escalation Rules

```
ORANGE Trigger: Resistance <2×/week for 1 week OR mobility <5min/day for 3+ days
  → ORANGE ADVISORY issued
  → Schedule minimum 2 sessions this week
  → Reduce sitting: stand every 25min (automatic with posture resets)
  → If injury: mobility work can substitute strength until cleared

ORANGE Trigger: ROM deficit >15° vs contralateral
  → Starrett pre-rehab protocol mandatory 2×/day
  → No new exercises targeting that joint until ROM restored
  → Physical therapy referral if deficit persists >4 weeks

RED Trigger: Resistance = 0×/week OR joint injury preventing all training
  → HARD FREEZE if 2+ weeks without resistance training
  → Focus on mobility only during RED recovery
  → Beth notification if correlated with other LD03 ORANGE signals
  → No Zone 5 sprints until strength returns
```

## ORANGE Recovery Protocol

### Immediate Actions (within 24h)

```
1. If strength gap: schedule minimum 2 sessions this week (non-negotiable)
2. If mobility gap: add 10min morning mobility before shower
3. If posture gap: install standing desk reminder + posture app
4. If injury: implement modified protocol (mobility only, no load)
5. Eliminate sitting >50min without standing (automatic with resets)
6. Add walking (low-intensity Zone 2) to compensate for strength gap
7. Log recovery protocol and adherence
```
