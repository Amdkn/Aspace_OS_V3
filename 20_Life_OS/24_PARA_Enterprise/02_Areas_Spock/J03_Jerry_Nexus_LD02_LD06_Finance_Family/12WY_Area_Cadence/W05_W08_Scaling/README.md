### Focus

- Performance review vs. S&P 500 benchmark

- TIER 3 equity valuation update

### Key Actions

```
□ Rebalance TIER 2 if drift > 5%
□ Review expense ratios (target < 0.10%)
□ Calculate YTD return vs. benchmark
□ Identify tax-loss harvesting opportunities
□ Update TIER 3 equity valuations
□ Document H1 investment decisions for audit
```

### The Rebalancing Rule

```
If any single asset > 25% of portfolio:
  → Rebalance back to target allocation
  → Document the rebalance
  → Check for tax implications before selling
```

## Common Failure Modes

- **Cash sitting idle**: Deploying late costs compounding time
- **Skipping rebalancing**: Drift creates concentration risk
- **Ignoring underperformance**: Small problems become big problems
- **Not harvesting losses**: Wasted tax optimization

- Any ORANGE = pause non-essential deployment
- Capital deployment > $5,000 requires 72-hour wait + documentation
