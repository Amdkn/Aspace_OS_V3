# B3 — Artifact Proofs
## Financial Decision Audit Trail + Evidence Standards

---

## Purpose

Artifact Proofs are the **evidence layer** — documentation that financial decisions were made correctly, with proper analysis, and within established thresholds. They create the audit trail that proves J03 is operating within its mandate.

---

## Required Artifacts

### 1. Runway Snapshots (Monthly)

```
Artifact: RUNWAY_YYYY-MM.md
Contents:
- Cash on hand: $____
- Monthly burn: $____
- Runway months: ___
- Gate status: 🟢/🟡/🔴
- Screenshot of account balances
```

### 2. Coverage Ratio Logs (Monthly)

```
Artifact: COVERAGE_YYYY-MM.md
Contents:
```

---

## Evidence Standards

### Minimum Documentation Requirements

For ANY capital allocation decision > $500:
1. Written analysis (even if brief)
2. Gate status at time of decision
3. Expected outcome documented
4. 30-day follow-up recorded

For ANY capital allocation decision > $5,000:
1. Full written analysis
2. Multiple scenarios (upside/downside)
3. Gate status + coverage ratio
4. Advisor consultation noted
5. 30-day AND 90-day follow-up

For ANY expansion during ORANGE:
1. Explicit A0 approval documented
2. Recovery plan attached
3. Timeline to GREEN specified
