## 3. MUSE-Eligible Criteria Summary

MUSE = Minimum Usable Solarpunk Experiment

A contribution is MUSE-eligible if it satisfies ALL 7:

| # | Rule | Test |
|---|------|------|
| 1 | Solarpunk thesis alignment | Does this advance regenerative rather than extractive systems? |
| 2 | Public goods character | Does this create value others can use without exclusion? |
| 3 | Relational capital growth | Does this deepen trust or build new connections? |
| 4 | Experiential capital deepening | Does this produce genuine engagement, not just activity? |
| 5 | Biomimetisme presence | Is nature-inspired design in the approach, not just output? |
| 6 | Circular thinking | Does this reduce waste or transform waste into input? |
| 7 | Knowledge sharing intent | Does this contribute to shared knowledge, not private advantage? |

A contribution is NOT MUSE-eligible if ANY disqualifier is present:
- Extraction primary (private profit with incidental public benefit)
- Green veneer ("sustainable" without circular design)
- Relational exploitation (uses relationships for extraction)
- Play deprivation (demands contribution without genuine play)
- Knowledge hoarding (creates knowledge but locks it)
