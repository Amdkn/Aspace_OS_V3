## 1. Domain Identity

### What This Domain OWNS

Experiential Capital is the flow artifacts layer — games, experiences, and play structures that produce authentic fun and flow, not manufactured engagement. It designs and tracks the experiential output that distinguishes solarpunk business from extraction machines.

### What This Domain Does NOT OWN

- Relational asset creation (that's D01)
- Network activation protocols (that's D02)
- LD08 doctrine (that's D05-D07)
- Contribution architecture (that's D08)

### Contribution Type

Experiential Capital contributes **flow artifacts** — experiences designed with the MDA framework (Mechanics, Dynamics, Aesthetics) that produce genuine play and deep engagement.

## 4. Flow State Science

Flow is not a reward. Flow is a **design material**.

| Flow Component | Design Rule |
|----------------|-------------|
| Clear goals | Every session must have a stated objective before it begins |
| Immediate feedback | Every action must produce a visible or felt result within 1 cycle |
| Challenge-skill balance | Target zone: 4% above current capability |
| Deep involvement | Eliminate competing attention before entering flow space |
| Sense of control | Agency must be real — pseudo-control does not produce flow |
| Loss of self-consciousness | Remove evaluation from the session — no one is watching |
| Transformation of time | Flow sessions should be time-boxed, not open-ended |

---

## 5. MDA Framework (Mechanics, Dynamics, Aesthetics)

| Layer | Definition | Design Target |
|-------|------------|---------------|
| **Mechanics** | Rules and structures | Should serve the dynamics |
| **Dynamics** | Player behavior the mechanics produce | Should serve the aesthetics |
| **Aesthetics** | Emotional experience the player has | Must be intentionally chosen |

**Anti-pattern**: Do not gamify what should not be gamified. Points, badges, and leaderboards applied to intrinsically motivating activities destroy intrinsic motivation.

---

## 6. True Fun vs False Fun Matrix

| Fun Type | Characteristics | Example | Contribution Test |
|----------|----------------|---------|-------------------|
| **True Fun** | Emerges naturally; intrinsically rewarding; enhances reality; freely chosen | Solving hard problem with friends; building something that matters | If activity stopped, would you miss it? Would work product be richer? |
| **False Fun** | Compensates for deprivation; requires external triggers; numbs; performed | Drinking to celebrate; scrolling to relax; forced play that feels like obligation | If activity stopped, would you feel relief? Is "fun" actually rest from deprivation? |

## 7. Superbetter Quest Protocol

| Phase | Protocol | Application |
|-------|----------|-------------|
| **Bandage** | Survive the crisis | Acknowledge the hit; do not minimize or magnify |
| **Quest** | Identify next milestone | "What is smallest next step that advances the work?" |
| **Ally** | Find the helper | Who has done this before? Who can witness your struggle? |
| **Bad Guy** | Name the obstacle | "The real problem is..." — be specific, not vague |
| **Power Up** | Recall past wins | "Remember when you handled X? You can handle this too." |

---

## 8. MUSE-Eligible Context

Experiential Capital is MUSE-eligible when:
- Experience produces genuine play, not engagement metrics
- Voluntary participation is non-negotiable
- "Win" condition is meaningful (victory without dignity is hollow)
- The experience advances solarpunk objectives, not just engagement
