## 1. Domain Identity

### What This Domain OWNS

Network Activation is the trust deployment layer — protocols for activating relational capital toward solarpunk objectives, not purely commercial ones. It operationalizes the giver-first protocols, Cialdini principles, and coalition-building strategies.

### What This Domain Does NOT OWN

- Relational asset creation (that's D01)
- Experiential design (that's D03/D04)
- LD08 doctrine (that's D05-D07)
- Contribution architecture (that's D08)

### Contribution Type

Network Activation contributes **trust deployment** — the strategic use of relational capital to build coalitions, form partnerships, and activate networks for solarpunk objectives.

## 2. LD05 Cialdini 6 Principles of Action

| Principle | Application |

## 3. Jerry's Stewardship

### When Relational Capital Activation is REQUIRED

Relational capital (LD05) must be deployed for solarpunk objectives when:
- A project involves cross-domain collaboration (requires trust-building from scratch)
- A partnership offers asymmetric benefit (one party gives more — feature, not bug, if giver is credible)
- A public contribution requires coalition-building (single contributors lack reach)
- A new market has no existing trust infrastructure (start with giver position, not extractor)

### When Relational Capital is NOT the Priority

Relational capital deployment is NOT required when:
- Transactional revenue activity — pure commerce with no public contribution dimension
- Internal operational execution — process without external partnership dimension
-已有信任基础的延伸 — extending existing relationships within proven network

### Activation Protocol

1. **Map the network first** — identify who has trust, who needs trust, who has already contributed to solarpunk commons
2. **Lead with giving** — activate with assets, not asks. Offer relational capital before requesting it.
3. **Use the right language** — match activation language to partner's love language and trust style
4. **Document the give** — track what was contributed, to whom, and when, even if no immediate return expected
5. **Follow through radically** — in solarpunk networks, reputation is the only currency that compounds

Network Activation is MUSE-eligible when:
