## 🧠 Concepts Clés à Haute Densité Sémantique

- **<Webvan (1996) — Too Early Trap>** : Louis Borders (Borders bookstore) commit 1Md$ pour warehouses automatisés Bechtel, fleet de camions, 26 centres de distribution. Brûlé 800M$ avant demande. Idée bonne, timing trop tôt (pas d'iPhone, pas de gig economy, dial-up). Lesson = 'right but too early' = pire que 'wrong'. On continue à croire, à dépenser, à noyer dans une vague qui n'est pas encore arrivée.
- **<Instacart (2012) — Webvan Done Right>** : Apoorva Mehta (ex-Amazon supply chain), 20 startups échouées. Slide d'intro = 'Webvan done right'. Pas de warehouse, pas de trucks, partenariat avec grocery stores existantes + shoppers indépendants via app. Slide d'intro = 'Webvan done right' (affronte le fantôme). Meh ta devenu YC summer 2012 batch via livraison 6-pack beer au partner YC.
- **<Skill Test (Mauboussin) — Luck vs Skill>** : Pure chance = tu peux pas perdre exprès (roulette). Venture = tu peux perdre exprès (mauvais founders → out en 1 an). Donc skill existe. Skill = pas contrôler la vague mais 'être dans l'eau avec la planche' = multiplier les chances que la luck te trouve.
- **<3 Leviers Founders — Preparation, Optionality, Pivoter>** : (1) Preparation = expertise domaine (Mehta a vécu Amazon supply chain → clock le smartphone + gig economy moment), (2) Optionality = burn bas + default alive (Webvan a fait l'inverse), (3) Pivoter willingness (Slack = failed game Glitch devenu chat interne devenu produit). Pattern = capital survival > capital maximization.
- **<Vintage Years — Fonds Timing>** : Carta data : meilleurs fonds VC landent juste après bulle pop (2010-2011 fantastiques). Fonds raised top of 2021-2022 trailing badly. Lowercase Ventures (Sacca, 2010, 8M$ seed) → Uber + Twitter + Instagram + Docker = >200× return. Lesson = timing de fonds > qualité individuelle du GP.
- **<Y Combinator = Machine à Fabriquer de la Sérendipité>** : YC concentre top founders + top capital + top network dans un même batch. 2009 batch = ~2500× return (Airbnb + Stripe). Lesson = la sérendipité n'est pas un accident, c'est un système construit pour maximiser la chance de rencontre.
