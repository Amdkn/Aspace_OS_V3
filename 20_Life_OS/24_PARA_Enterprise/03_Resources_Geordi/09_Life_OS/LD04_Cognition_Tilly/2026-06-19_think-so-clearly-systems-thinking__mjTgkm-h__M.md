# Think SO CLEARLY — Systems Thinking (Sandeep Swadia)

> **Intention** : diagnostiquer POURQUOI un système échoue (vs comment le fixer) — framework 4 types de systèmes + cobra effect + delayed feedback loops.

## 🎯 5 points clés

1. **Systems thinking** : "ability to see hidden patterns from parts you can observe" — connected parts producing a pattern.
2. **4 system types** : Clear (checklists) → Complicated (need experts) → Complex (probe-sense-respond) → Chaotic (act-sense-respond).
3. **3 reasons why systems fail** : don't know system type + cobra effect (incentive misalignment) + delayed feedback loops.
4. **M&M's test (Van Halen)** : 1 détail technique manquant dans un contrat = tout le système est suspect → "find your own M&M's".
5. **Cobra effect** : bounty sur cobras mortes → gens élèvent cobras pour toucher la bounty → plus de cobras qu'avant. Inapplicable aux incentives mal calibrés.
