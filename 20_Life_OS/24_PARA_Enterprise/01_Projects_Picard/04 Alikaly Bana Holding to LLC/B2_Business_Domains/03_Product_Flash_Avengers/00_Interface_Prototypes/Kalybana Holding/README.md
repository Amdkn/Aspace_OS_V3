## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- Gemini API Key

### Installation

To set up both platforms:

```pwsh
# Install Real Estate dependencies
cd 01_Real_Estate
npm install

# Install Holding Platform dependencies
cd ../02_Holdings_Platform
npm install
```

## 🛠️ Environment Configuration

Ensure `.env.local` is configured in each sub-directory with your `GEMINI_API_KEY`.
