# 03 Product - Flash / Avengers

## Role

Own interface prototypes, user workflows, product requirements, UX, demo surfaces, and build artifacts.

## Gate

Product may prototype freely, but cannot ship alone. Product Done means demoable; Business Done means gated by the other B2 domains.

## Required Input From Product

Must provide build notes, screenshots, workflows, scope boundaries, assumptions, and release candidate evidence to the other B2 domains.

## Blocking Authority

Blocks the project only when the core user value is not demonstrable or the interface does not match the promise.

## Operating Rule

This domain is not decorative. If this README has no gate status for the active release, the project remains PRODUCT_ONLY_PROTOTYPE.
