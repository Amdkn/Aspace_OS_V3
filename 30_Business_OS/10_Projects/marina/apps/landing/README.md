# Marina Cleaning Front End

Next.js front end for the Marina Super Cleaners landing page.

## Source Merge

This folder merges:

- `00_Interface_Prototypes/marina-cleaning-frontend` as the active Next.js app.
- `00_Interface_Prototypes/Marina Super Cleaners (Copy)` under `legacy_sources/` for historical HTML, audit, and scaffold evidence.

The active app lives at the repository root. Legacy sources are preserved for traceability and should not be used as the production runtime.

## Commands

```powershell
npm run lint
npm run build
npm run dev
```
