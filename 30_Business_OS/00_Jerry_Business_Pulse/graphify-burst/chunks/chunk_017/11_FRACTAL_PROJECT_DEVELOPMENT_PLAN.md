# Fractal Project Development Plan - 00 Agency as a Service

## Purpose

This file turns Jerry's macro Business Wheel into concrete Summer project development inside this project folder.

## Project Thesis

AaaS is a flagship agency-as-a-service offer stack that must prove one sellable flagship offer, one expansion offer, and zero-Jerry delivery.

Primary mode: **Solaris/Nexus/Orbiter portfolio**.

Core pain: **clients need outcomes without a fragile bespoke agency dependency**.
